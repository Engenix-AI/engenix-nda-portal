CLOUDFLARE PAGES SETUP

1. Deploy this package as the Pages project root/output.
2. Confirm this path exists in the deployed project:
   functions/api/demo-request.js
3. In Cloudflare Pages:
   Settings → Variables and Secrets

Add:
   RESEND_API_KEY       your Resend API key, stored as a secret
   DEMO_TO_EMAIL        example: nicholas@engenix.co
   DEMO_FROM_EMAIL      example: ENGENIX Website <website@engenix.co>

Optional:
   SEND_CONFIRMATION_EMAIL = true
   Set to false to disable requester confirmations.

4. In Resend, verify engenix.co and the sending address/domain.
5. Redeploy after saving variables.
6. Test both:
   /beta/ Private Access drawer
   /beta/demo.html

IMPORTANT FOR /beta/
The HTML can live under /beta/, but the functions directory must be deployed at the Cloudflare Pages project root so this endpoint resolves:
   /api/demo-request
